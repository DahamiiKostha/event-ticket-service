Control Panel

Use the buttons below to turn the simulation Start or Stop. The current status will be displayed in real-time.

What is this Simulation?

This simulation allows you to control an interactive system where the button states represent different modes of operation. You can explore dynamic changes by toggling between "Start Simulation" and "Stop Simulation"

    Start Simulation: Enables the simulation and activates interactive features.
    Stop Simulation: Disables the simulation and pauses all processes.
